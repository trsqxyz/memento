The memento outputs "*" every 5 minutes for 5 times.(default)  

# Usage
`python -m memento`  
or  
`python -m memento mori 11` # outputs for 12 times.(1 hour)  
or  
`python -m mmemento pomodoro`

# mode  
- mori(memento mori)
